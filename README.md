# 超市管理系统 java648

## 项目概述

本项目是基于SpringBoot和Vue的超市管理系统，适用于零售门店管理，实现了超市进销存管理以及智能收银功能。通过一体化数字处理，提高了运营效率和管理水平。

## 技术栈

### 架构
- B/S结构
- MVC软件架构
- 前后端分离架构

### 后端
- Spring Boot
- MyBatis Plus
- Lombok
- RBAC

### 前端
- Vue
- ECharts
- Element UI
- Axios

### 数据库
- MySQL
- Redis

### 开发环境
- IDEA
- JDK
- Maven
- Node

## 功能模块

系统包括以下功能模块：
- 会员管理
- 个人中心
- 人事管理
- 销售管理
- 库存管理
- 系统管理
- 商品管理

系统角色包括：
- 系统管理员
- 仓库管理员
- 销售人员

## 系统亮点
- 前后端分离的架构设计
- Redis数据库缓存应用
- 基于RBAC的多角色访问控制
